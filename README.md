# K1SS — Instruments de musique modulaires open source

**Assemblé en Franche-Comté · Open hardware CERN-OHL-S · Firmware GPL**

---

K1SS est un écosystème d'instruments de musique modulaires, open source, assemblés en France. Le concept : un hub central (B0DY) sur lequel se connectent des modules créatifs interchangeables (H34D / N3CK).

## Produits

| Produit | Description |
|:--------|:------------|
| **B0DY** | Hub central, carte son XMOS professionnelle, connecte jusqu'à 4 modules |
| **H34D** | Modules créatifs (séquenceur, granulaire, effets, tactile…). 250+ designs |
| **N3CK** | Modules légers coprocesseurs |
| **SM4LL** | Standalone — prêt à l'emploi, sans hub |
| **F00T1** | Socle interne, jamais vendu seul |
| **H4T Club** | Abonnement communauté — vote et proposition de designs |
| **H4T Store** | Apps firmware pour chaque module |
