# K1SS — Information for Investors

**K1SS** (K1 Saillard Saillard) builds open-source modular musical instruments.  
The first product, **D1J1**, is the first hardware instrument designed natively for [Mixxx](https://mixxx.org) — the leading open-source DJ software with ~2 million active users.

This document is for prospective investors, partners, and anyone who wants to understand the project beyond the product surface.

---

## Why we exist

Electronic music hardware has been dominated by the same closed ecosystems for 20+ years. Mixxx has ~2 million users and zero hardware designed for it. Controllers are proprietary, unrepairable, and built for software that costs as much as the hardware itself.

We build differently: open source, community-driven, assembled in France, designed to last.

## The opportunity

- **Mixxx ecosystem**: ~2M active users, zero native hardware, first-mover advantage
- **Open hardware wave**: right to repair, transparency, and maker culture are growing
- **Community-driven design**: users vote on what gets built — lower R&D risk, built-in demand
- **Local manufacturing**: Franche-Comté, France — short supply chain, traceable components
