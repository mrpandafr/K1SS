# Product Overview

## D1J1 — First Mixxx-native hardware

D1J1 is a live performance instrument combining DJ controls, a standalone synthesizer, and wireless connectivity. It is the first hardware device designed specifically for Mixxx, the leading open-source DJ software with ~2 million users.

### Key features

- **Native Mixxx integration** — controls map directly to Mixxx's architecture, not retrofitted via MIDI mapping
- **Standalone synth mode** — works without a computer
- **WiFi MIDI Link** — wireless multi-device synchronization via Ableton Link
- **5" display**, 12 RGB pads, 4-output sound card
- **Ambidextrous design** — left-handed and right-handed configurations
- **Open source** — CERN-OHL-S hardware, GPL firmware
- **Assembled in France** — Franche-Comté, short supply chain

## M4Z3 Ecosystem

D1J1 is the entry point to the M4Z3 ecosystem: an open modular platform where a central unit connects to interchangeable modules.

| Component | Description |
|:----------|:------------|
| **Hub** (2028) | Central unit: audio, power, connectivity, DSP |
| **Modules** | Interchangeable: sequencers, effects, granular synths, tactile surfaces |
| **Community catalog** | Users vote on which modules get manufactured next |

### Design principles

- **Right to repair**: full schematics published, no proprietary connectors, no firmware lock
- **No cloud dependency**: all instruments work without a server or subscription
- **Modular longevity**: replace one module, not the whole instrument
- **Open contribution**: anyone can build, modify, or fork under CERN-OHL-S
